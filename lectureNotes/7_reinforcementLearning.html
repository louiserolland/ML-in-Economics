Reinforcement Learning

Markov Process

Markov Reward Process

Return is the discounted sum of rewards (net present value): \[\begin{align*} G_t = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \end{align*}\]

Markov Decision Process

Partially Observable MDP (POMDP)

Policy \(\pi(a_t|s_t)\)

Value Function: Expected Return

  • State-value function \(V^{\pi}(s_t)=\mathbb{E}[G_t|s_t,\pi]\)
  • State-action value function \(Q^{\pi}(s_t,a_t)=\mathbb{E}[G_t|s_t,a_t,\pi]\)
  • Optimal state-value funtion \(V^*(s_t)=\max_{\pi}V^{\pi}(s_t)\)
  • Optimal state-action value function \(Q^*(s_t,a_t)=\max_{\pi}Q^{\pi}(s_t,a_t)\)
  • Optimal Policy \(\pi^*(a_t|s_t)\leftarrow \arg\max_{a_t}Q^*(s_t,a_t)\)

Bellman Equation

\[\begin{align*} V^{\pi}(s_t) &= \sum_{a_t} \pi(a_t|s_t) Q(s_t,a_t) \end{align*}\]

\[\begin{align*} Q^{\pi}(s_t,a_t) &= r(s_t,a_t) + \gamma \sum_{s_{t+1}} P(s_{t+1}|s_t,a_t) V^{\pi}(s_{t+1}) \end{align*}\]

and therefore

\[\begin{align*} V^{\pi}(s_t) &= \sum_{a_t} \pi(a_t|s_t) \left[ r(s_t,a_t) + \gamma \sum_{s_{t+1}} P(s_{t+1}|s_t,a_t) V^{\pi}(s_{t+1}) \right]\\ Q^{\pi}(s_t,a_t) &= r(s_t,a_t) + \gamma \sum_{s_{t+1}} P(s_{t+1}|s_t,a_t) \sum_{a_{t+1}} \pi(a_{t+1}|s_{t+1}) Q^{\pi}(s_{t+1},a_{t+1}) \end{align*}\]

Model-Based (MB) vs. Model-Free (MF)

  • Model-based: \(P(s_{t+1}|s_t,a_t)\) and \(r(s_t,a_t)\) are known
    • Dynamic programming
  • Model-free: no model of \(P(s_{t+1}|s_t,a_t)\) and/or \(r(s_t,a_t)\)
    • Value estimation: estimate \(Q^*(s_t,a_t)\) and derive \(\pi^*(a_t|s_t)\)
      • Monte Carlo value estimation
      • Temporal difference (TD) methods
    • Policy estimation: estimate \(\pi^*(a_t|s_t)\) directly (using gradient descent)

Tabular Reinforcement Learning

(Prince 2023, Ch. 19.3)

Tabular RL algorithms do not rely on function approximation.

Dynamic Programming (MB)

\[\begin{align*} V(s_t) &\leftarrow \sum_{a_t} \pi(a_t|s_t) \left[ r(s_t,a_t) + \gamma \sum_{s_{t+1}} P(s_{t+1}|s_t,a_t) V(s_{t+1}) \right] \\ \pi(a_t|s_t) &\leftarrow \arg\max_{a_t} \left[ r(s_t,a_t) + \gamma \sum_{s_{t+1}} P(s_{t+1}|s_t,a_t) V(s_{t+1}) \right] \end{align*}\]

Policy iteration: At each policy improvement, iterate policy evaluation to convergence.

Value iteration: One policy evaluation step per policy improvement.

Monte Carlo Value Estimation (MF)

On-policy: Alternate between estimating \(Q(s,a) \approxeq \frac{1}{N}\sum_e G_t(s,a,\pi)\) from a series of episodes \(e\), and updating policy according to \[\begin{align*} \pi(a|s) \leftarrow \arg\max_a Q(s,a) \end{align*}\]

Off-policy: Target policy \(\pi\) learned with episodes from behavior policy \(\pi'\).

Temporal Difference (TD) Learning (MF)

Combine bootstrapping of dynamic programming with sampling of MC, updating values & policy while agent interacts with environment.

SARSA (on-policy): \[\begin{align*} Q(s_t,a_t) &\leftarrow Q(s_t,a_t) + \alpha \left[ r(s_t,a_t) + \gamma Q(s_{t+1},a_{t+1}) - Q(s_t,a_t) \right] \end{align*}\]

Q-learning (off-policy): \[\begin{align*} Q(s_t,a_t) &\leftarrow Q(s_t,a_t) + \alpha \left[ r(s_t,a_t) + \gamma \max_{a} Q(s_{t+1},a) - Q(s_t,a_t) \right] \end{align*}\]

Updates are contraction mappings so \(Q(s,a)\) converges.

Deep Q-Learning (MF)

(Prince 2023, Ch. 19.4)

Fitted Q-Learning

ML model \(q(\mathbf{s}_t,a;\phi)\) where \(\mathbf{s}_t\) is now a state vector.

\[\begin{align*} \phi \leftarrow \phi + \alpha \left[ r(s_t,a_t) + \gamma \max_{a} q(s_{t+1},a;\phi) - q(s_t,a_t;\phi) \right] \nabla_{\phi} q(s_t,a_t;\phi) \end{align*}\] to minimize the least squares loss \[\begin{align*} L(\phi) = \left( r(s_t,a_t) + \gamma \max_{a} q(s_{t+1},a;\phi) - q(s_t,a_t;\phi) \right)^2 \end{align*}\]

Convergence no longer guaranteed.

Deep Q-Network (DQN)

  • Experience replay: batch sampled from replay buffer of recent \((s_t,a_t,r_t,s_{t+1})\) tuples.
  • \(\max_{a} q(s_{t+1},a;\phi^{-})\) with \(\phi^{-}\) updated only periodically

Policy Gradient Methods (MF)

(Prince 2023, Ch. 19.5)

Gradient Update

Directly learn policy \(\pi(a_t|\mathbf{s}_t;\mathbf{\theta})\) \[\begin{align*} \arg\max_{\mathbf{\theta}} \int \left(\sum_{t=1}^T r(\mathbf{s}_t,a_t)\right) P(\mathbf{s}_1)\prod_{t=1}^T \pi(a_t|\mathbf{s}_t;\mathbf{\theta})P(\mathbf{s}_{t+1}|\mathbf{s}_t,a_t) d\mathbf{s}_1da_1 \ldots d\mathbf{s}_T d a_T \end{align*}\] yields (approx. integral with sum over \(I\) episodes) gradient ascent update \[\begin{align*} \mathbf{\theta} &\leftarrow \mathbf{\theta} + \frac{\alpha }{I}\sum_{i=1}^I\sum_{t=1}^T \nabla_{\mathbf{\theta}} \log \pi(a_{it}|\mathbf{s}_{it};\mathbf{\theta}) \sum_{k=t}^T r_{ik} \end{align*}\]

Actor-Critic Methods

Temporal difference (TD) policy gradient method.

\[\begin{align*} \theta &\leftarrow \theta + \frac{\alpha }{I}\sum_{i=1}^I\sum_{t=1}^T \nabla_{\theta} \log \pi(a_{it}|\mathbf{s}_{it};\theta) \left(r_{it} + \gamma v(\mathbf{s}_{i,t+1},\phi) - v(\mathbf{s}_{i,t},\phi) \right)\\ L(\phi) &= \sum_{i=1}^I\sum_{t=1}^T \left( r_{it} + \gamma v(\mathbf{s}_{i,t+1},\phi) - v(\mathbf{s}_{i,t},\phi) \right)^2 \end{align*}\] where \(\pi(\mathbf{s}_t,\mathbf{\theta})\) is the actor and \(v(\mathbf{s}_t,\phi)\) the critic.

Offline RL: Decision Transformer

Replace reward \(r_t\) with returns-to-go \(R_{t:T}=\sum_{t'=t}^T r_{t'}\) during training.
